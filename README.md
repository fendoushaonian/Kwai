# Kwai
"Kwai" 是刘鸿运开发的一款Python 快手库，可以调用快手基本功能，列如 : 快手关注，快手举报，快手检测账号是否被拉黑，获取别人快手IP,快手获取评论区,快手账号转User_ID等等，功能很多，以后会更新调用快手智能小快，提供逆向JS，补好的环境，提供快手算法等等，目前是Kwai 1.0版本，提供公测，测试 !
